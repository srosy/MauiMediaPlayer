# Security

If you believe you have found a security vulnerability in this project, please report it responsibly.

- Prefer [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) for this repository (private disclosure).
- Do not open a public issue for undisclosed security-sensitive bugs.

Maintainers will work with you to understand and address the issue.
